# ILI9341_XPT2046

## Language

Spin2, PASM2

## Category

Display

## Description

ILI9341_XPT2046 is a spin2 object that interfaces the ILI9341 Video chip and the XPT2046 chip with the Propeller 2 P2X8C4M64P

Learn more about the Propeller 2 at [www.parallax.com](http://www.parallax.com).
