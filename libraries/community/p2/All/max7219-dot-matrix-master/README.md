# MAX7219 Dot Matrix 

## Language

Spin2

## Category

Display

## Description

MAX7219 Dot Matrix is a spin object that interfaces the MAX7219 chip with the Propeller 2 P2X8C4M64P

Learn more about the Propeller 2 at [www.parallax.com](http://www.parallax.com).
